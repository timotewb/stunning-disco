# stunning-disco

A sandbox repository for storing and versioning resources across various technologies and use cases.

## RAG using langchain and ollama

Credit: [link](https://medium.com/@isurulkh/simple-rag-app-using-ollama-and-langchain-96ea10bf79ce)

### Project Structure

- chain_handler.py # Setup retrieval and question-answering chain
- doc_retrieval.py # Document retrieval functionality
- docs_db_handler.py # Document loading and vector store management
- embedder.py # Embedding model setup
- indexer.py # Document splitting
- retriever.py # Document retrieval
- session_handler.py # Session history management
- webui.py # Streamlit web interface
